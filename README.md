# Java-Pratice
This is my first repo to restore my java pratice code

## Structure

```
src/
├── basics/
│   ├── HelloWorld.java        - Classic Hello World program
│   ├── DataTypes.java         - Java primitive data types demo
│   └── ControlFlow.java       - if/else, switch, for, while, do-while
├── datastructures/
│   ├── ArraysAndCollections.java  - Arrays, sorting, ArrayList
│   └── HashMapExample.java        - HashMap operations
└── oop/
    ├── Inheritance.java       - Classes, subclasses, polymorphism
    └── Interfaces.java        - Interface definition and implementation
```

## Running the Examples

Compile a file:
```bash
javac src/basics/HelloWorld.java
```

Run a program:
```bash
java -cp src/basics HelloWorld
```
